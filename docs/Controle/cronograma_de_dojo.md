# Cronograma de Dojos

## Contexto 

Com o intuito de nivelar o conhecimento relacionado às possíveis tecnologias a serem utilizadas, foi criado um cronograma para aplicação de Dojos, compreendendo os seguintes assuntos:

- Linux
- Git
- HTML/CSS (Web)
- Javascript
- Métodos Ágeis
